## Laravel Todo Api

Laravel todo api is a basic CRUD service for creared tasks.

## Setup

- Clone the repository

- Run migrations
```bash
php artisan migrate
```

- Run seeders
```bash
php artisan db:seed --class=TaskSeeder
```

- Start Server
```bash
php artisan serve
```
